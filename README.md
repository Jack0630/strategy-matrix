# 策略矩陣分析工具 Strategic Matrix Analyzer

> 「策略矩陣」（Strategic Matrix）是由政大管理學大師司徒達賢教授提出的分析工具，
> 主要用於將企業複雜的競爭策略轉化為直觀、結構化的視覺地圖。

🔗 **線上使用：** [https://jack0630.github.io/strategy-matrix/](https://jack0630.github.io/strategy-matrix/)

---

## ✨ 功能介紹

### 🎨 策略矩陣繪製
- 點選格子畫箭頭，自由描繪競爭者在各價值活動的定位與走向
- 支援最多 8 位競爭者，每位使用不同顏色區分
- 箭頭可刪除，支援復原（最多 80 步）

### 🏷️ 自訂欄位
- 欄位名稱可直接編輯
- 支援拖曳排序
- 可新增或刪除欄位

### 🎨 配色主題
5 種主題一鍵切換：紫藍、青綠、靛藍、玫瑰、暖橙

### 📤 多種匯出格式
| 格式 | 說明 |
|------|------|
| 📥 匯出 PNG | 匯出完整策略矩陣圖片 |
| 🎬 匯出動畫序列 | 逐步揭露的多張 PNG，可自行排進 PPT |
| 📊 匯出 PPTX | 直接產生 PowerPoint，含逐步動畫 + 完整總覽投影片 |

---

## 🚀 使用方式

### 基本操作
1. **填入策略名稱與說明**（選填）
2. **設定競爭者名稱**，點擊 chip 上的名稱可直接改名
3. **切換到繪製模式**，點第一格（起點）→ 點第二格（終點）→ 自動畫出箭頭
4. 重複步驟 3，繪製完整策略路徑

### 匯出動畫 PPT
1. 選擇要匯出的競爭者（點選對應 chip）
2. 點「📊 匯出 PPTX」
3. 自動產生逐步揭露的投影片，最後一頁為完整策略圖
4. 在 PowerPoint 中按 F5 播放即可

## 📐 矩陣結構

| 維度 | 說明 |
|------|------|
| **欄（活動）** | 原料、零組件、採購、研發、製選、資訊系統、倉儲、運輸、通路、品牌、產品一、產品二、產品三（可自訂） |
| **列（策略維度）** | A 產品線廣度與特色、B 目標市場區隔與選擇、C 垂直整合程度之取決、D 相對規模與規模經濟、E 地理涵蓋範圍、F 競爭優勢 |

## 🛠️ 技術說明

- 純前端單頁應用（Single HTML File）
- 無需後端、無需安裝，直接用瀏覽器開啟即可使用
- 使用技術：HTML / CSS / JavaScript、SVG、html2canvas、JSZip
- 支援瀏覽器：Chrome、Edge、Safari、Firefox（建議使用 Chrome）
